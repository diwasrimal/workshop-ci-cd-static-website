Cloud Computing & DevOps Fundamentals

50-day AWS-Focused Workshop Summary

About This Workshop

This 50-day workshop introduced core cloud computing concepts, AWS services, networking, DevOps culture, CI/CD pipelines, monitoring, and deployment practices.

Day 1

Cloud Foundations & AWS Intro

  • Traditional vs Cloud Computing
  • IaaS, PaaS, SaaS
  • Public, Private, Hybrid Cloud
  • AWS Regions & Availability Zones
  • IAM Users, Roles & Policies

Day 2

AWS Core Services

  • Amazon EC2 Basics
  • Security Groups & Key Pairs
  • Amazon S3 Storage
  • SQL vs NoSQL
  • RDS & DynamoDB

Day 3

Networking & DevOps

  • IP Addressing & Subnets
  • Amazon VPC
  • Public vs Private Subnets
  • Security Groups vs NACL
  • DevOps Lifecycle

Day 4

Git & CI/CD

  • Git & GitHub Basics
  • Clone, Commit, Push
  • Branching Workflow
  • Containers & Orchestration
  • CI/CD Pipelines

Day 5

Monitoring & Deployment

  • CloudWatch Monitoring
  • Metrics vs Logs
  • AWS Lambda
  • CloudFront & SNS
  • Mini AWS Project

Workshop Outcome

Participants gained practical knowledge of AWS cloud services, DevOps tools, automation pipelines, and deployment strategies.
